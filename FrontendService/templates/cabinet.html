SHP – Simple Hide to PDF

Привет, пользователь бескрайнего рунета!

Перед вами Just4Fun-проект по сокрытию данных внутри PDF документов.
С помощью данного портала вы можете скрывать текстовую информацию внутри PDF документов
(это называется стеганографией)

Схема работы очень простая:

Вкрапление

  1. Придумайте пароль (и запомните его!)
  2. Напишите сообщение для сокрытия
  3. Выберете pdf документ для вкрапления информации
  4. Нажмите кнопку вкрапить
  5. Полученный документ – это файл с вкрапленным сообщением

Извлечение

  1. Введите пароль (тот самый, который запомнили)
  2. Выберете pdf документ со скрытым сообщением
  3. Нажмите кнопку извлечь
  4. Если пароль верен и документ содержит скрытый текст, система выдаст вам сокрытое сообщение
